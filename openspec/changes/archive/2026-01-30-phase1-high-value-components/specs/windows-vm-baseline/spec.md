### Requirement: Windows baseline placeholder
The system SHALL document external artifacts required for Windows VM provisioning (Windows ISO, virtio ISO) and how they are referenced via `.env` paths.

#### Scenario: External artifacts documented
- **WHEN** the Windows baseline spec is reviewed
- **THEN** it lists required ISOs and states they must not be committed to git

#### Scenario: Acceptance tests marked TBD

- **THEN** it indicates acceptance tests are pending future implementation
